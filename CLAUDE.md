# Dresser

This is a very personal application for myself that serves a purpose: laundry tracking.
I live across two houses, I have various clothing articles, and I want to know where clothes are and if they are clean or dirty.

## Core Features

### Invariants

_Invariants_ are the core feature of this application. They are things about my laundry that **should** be true. If they are not true, I should fix them buy doing laundry or buying more clothes.

Examples:

- I must have a pajama shirt and pajama pants clean at each house
- I should have 3 pairs of underwear clean at each house
